About

Sehee's Mini Games 는 학습 겸 취미로 만든 정적 웹 기반 미니게임 모음입니다. 모든 게임은 별도 설치 없이 최신 브라우저에서 바로 실행되며, 한국어 인터페이스를 제공합니다.

오목대전과 2048 모두 Node.js · WebSocket 기반 실시간 서버를 사용합니다. 오목은 보드 진행·봇 대전·관전·랭킹을, 2048은 점수 등록과 전체/오늘 랭킹을 서버가 처리합니다. 제작 의도와 기술 스택은 GitHub 저장소 README에서 확인할 수 있습니다.
